Технические требования:
Язык TypeScript
Консольное приложение, т.е. UI - терминал (командная строка). Проверка корректности команда обязательна. 
Разрешено использовать любые фреймворки и библиотеки.
Необходимость построения абстракций остается на ваше усмотрение, но её наличие будет плюсом.

Пример взаимодествия:
Консоль: Команда "add product" - добавить продукт, "show list" - показать список введенных продуктов.
Пользователь: add list
Консоль: Введена неверная команда: add list.
Консоль: Команда "add product" - добавить продукт, "show list" - показать список введенных продуктов.
Пользователь: show list
Консоль: Список пуст
Пользователь: add product
Консоль: Введите тип продукта. Доступные типы: ice cream, soft drink
Пользователь: ice cream
Консоль: Введите название продукта
Пользователь: Delicious ice
Консоль: Введите вес единицы продукта в граммах
Пользователь: 150
Консоль: Товар добавлен
Консоль: Команда "add product" - добавить продукт, "show list" - показать список введенных продуктов.
Пользователь: add list
Консоль: Введите тип продукта. Доступные типы: ice cream, soft drink
